DataTable¶

See: https://material.io/components/data-tables

Data tables display information in a grid-like format of rows and columns. They organize information in a way that’s easy to scan, so that users can look for patterns and insights.

Data tables can contain:

  • Interactive components (such as chips, buttons, or menus)

  • Non-interactive elements (such as badges)

  • Tools to query and manipulate data

class materialweb.tags.data_table.DataTable(*args, **kwargs)¶

Provides template tag: DataTable.

Example usage:

{% load materialweb %}

{% trans "Dessert calories" as label %}
{% DataTable label=label %}
  {% DataTable_Head %}
    {% DataTable_Head_Row %}
      {% DataTable_Head_Col %}
        {% trans "Dessert" %}

      {% endDataTable_Head_Col %}
    {% endDataTable_Head_Row %}
  {% endDataTable_Head %}

  {% DataTable_Body %}
    {% DataTable_Row %}
      {% DataTable_ColHeader %}
        {% trans "Frozen yogurt" %}
      {% endDataTable_ColHeader %}

      {% DataTable_Col type="num" %}
        24

      {% DataTable_Col type="num" %}

        {% trans "Super tasty" %}

    {% endDataTable_Row %}

      {% DataTable_ColHeader %}
        {% trans "Ice cream sandwich" %}
      {% endDataTable_ColHeader %}

        37
      {% endDataTable_Col %}

      {% DataTable_Col type="num" %}
        4.33333333333
      {% endDataTable_Col %}

      {% DataTable_Col %}
        {% trans "I like ice cream more" %}
      {% endDataTable_Col %}
    {% endDataTable_Row %}

    {% DataTable_Row %}
      {% DataTable_ColHeader %}

      {% endDataTable_ColHeader %}

      {% endDataTable_Col %}

  {% endDataTable_Body %}
{% endDataTable %}
